# 유니티 포트폴리오( 랜덤 타워 디펜스 )

유니티로 제작한 랜덤타워 디펜스류 게임 포트폴리오 입니다.

포톤 으로 멀티플레이가 가능하게 제작하였습니다.

제작기간은 한달입니다. 사용 언어는 C#입니다.
구현된 되자인패턴 으로는 오브젝트 풀링, 전략패턴, 팩토리 패턴, 옵저버 패턴이 있습니다.

유니티 버전은 2019.4.21f 버전입니다.
